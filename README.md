# INSTALL

```sh
$ termux-setup-storage
$ pkg install update && pkg install upgrade
$ pkg install ffmpeg && pkg install bash && pkg install nodejs
$ git clone https://github.com/alanwildan/selfbot
$ cd selfbot
$ ls
$ bash install.sh
$ node index.js

Tinggal Scan Dah :)

Cara menampilkan menu : zhelp/zmenu
```
